# Eco Guide 

## Feature 

- List FAF units 
- Let user do cross comparsion between them 
- User can click the unit link to see its detailed infomation.


## Specifications
- For now, this is pure dioxus frontend project. We plan to deploy it on GitHub sites. No backend for now.
- Instead of store units in DB, let's hardcode units for now in frontend. 
- Use the same layout and visual experience as current phoenix app.
- For CSS, check https://github.com/DioxusLabs/dioxus/pull/5087 to see if the new feature could help you to organize CSS.
